Spring Security

SpringBoot + Spring Security + Thymeleaf

前置demo搭建

  • 创建一个简单的springboot项目
  • 依赖勾选spring security，spring web，thymeleaf，项目加载完先注释掉spring security相关的依赖，不然运行报错
  • 设置项目端口8080，resources/template/目录下创建index.html和views文件夹
  • views文件夹中继续创建三个文件夹分别是level1，level2，level3，每个文件夹下面创建三个html文件，前缀为1，2，3，在这些html文件中稍微写下标识表示一下是哪个文件夹下的文件页面
  • views文件夹下创建一个login页面
  • index页面包含登录注销链接，三个模块展示，第一个模块是level1页面的三个链接，点击进入对应level的html页面，其他模块相似；登录模块包括用户名，密码输入框和一个登录按钮；
  • 创建跳转页面的RouterController，代码放后面，运行测试页面跳转
  • 创建spring security配置文件SecurityConfig继承WebSecurityConfigurerAdapterweb安全配置适配器，重写一个授权方法和一个权限验证方法，主要的代码如下，后面大部分配置都是在这个文件里面配置的，具体功能看代码注释
  • 最主要的代码就是跳转页面类和配置类，写完所有相关代码后去掉spring security依赖的注释，可能还需要添加thymeleaf和spring security整合的依赖，运行
